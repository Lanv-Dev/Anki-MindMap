# Anki-MindMap

Anki Mind Map is a custom note type for AnkiDroid that allows users to create and edit mind maps directly within a note. Unlike external mind-mapping tools, the entire editing experience takes place inside AnkiDroid, making it easy to build and review knowledge without leaving the app.

The project is inspired by Anki's Image Occlusion note type, adapting its concepts to support interactive, editable mind maps. Each node in the map can also function as an individual flashcard, allowing specific concepts to be reviewed independently while remaining part of the overall map structure.

The implementation is built entirely with HTML and JavaScript, with no external dependencies. Everything runs inside the note type itself, making it lightweight, portable, and easy to customize.

Note: At the moment, editing is supported only in AnkiDroid. Full editing support for the desktop version of Anki is still a work in progress.
